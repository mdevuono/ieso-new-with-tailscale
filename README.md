# IESO LMP Viewer

A lightweight local web app for viewing IESO Day-Ahead and Predispatch Locational Marginal Price (LMP) data, with support for private confidential schedule reports. Served over Tailscale for secure access from any device on your tailnet.

---

## Features

- **Public LMP Reports** — Fetches DA and Predispatch Hourly Energy LMP CSVs directly from the IESO public report site for any date
- **Private Schedules** — Fetches confidential Predispatch Schedule XML reports from the IESO private report API using a machine account
- **Current HE Highlighting** — Automatically highlights the current Hour Ending in all tables based on IESO time (EST, UTC-5)
- **Auto-save** — All fetched files are saved to a local `data/` folder on each fetch
- **Tailscale Access** — Server binds to all interfaces so any device on your Tailscale network can access it
- **No manual download/upload** — All data is fetched server-side, bypassing browser CORS restrictions

---

## Prerequisites

- [Python 3.8+](https://www.python.org/downloads/) — check **"Add Python to PATH"** during installation
- [Tailscale](https://tailscale.com/download) — sign in with the same account on all devices
- IESO machine account credentials (for private reports) — contact IESO customer relations to request: `customer.relations@ieso.ca` / 1-888-448-7777

## Installation

1. Clone or download this repository
2. Run `install.bat` (right-click → Run as Administrator recommended)
3. Follow the prompts — installer will:
   - Verify Python is installed
   - Create `%USERPROFILE%\IESO_LMP_Viewer\` with a `data\` subfolder
   - Copy all application files
   - Install Python dependencies (`flask`, `requests`)
   - Prompt for your IESO machine account username and password, writing them to `config.py`
   - Create a desktop shortcut

> **Note:** If `config.py` already exists (e.g. on reinstall), the credential prompt is skipped. To update credentials, delete `config.py` and re-run `install.bat`.

---

## Running the Server

Double-click the **IESO LMP Viewer** desktop shortcut, or run:

```
Start_LMP_Server.bat
```

The Flask console window will open and your browser will automatically navigate to `http://localhost:5000`.

To **stop the server**, click the console window and press `Ctrl+C`.

---

## Accessing from Other Devices

With Tailscale running on both machines, open a browser on any device and navigate to:

```
http://<your-laptop-tailscale-ip>:5000
```

Find your Tailscale IP with:

```powershell
tailscale ip -4
```

```
├── install.bat              # One-time installer
├── Start_LMP_Server.bat     # Server launcher (daily use)
├── server.py                # Flask proxy server
├── index.html               # Web UI
├── .gitignore               # Excludes config.py and data/ from Git
├── config.py                # ⚠ Generated by install.bat — NOT in repo
└── data/                    # ⚠ Downloaded report files — NOT in repo
```

---

## Security Notes

- `config.py` contains plaintext credentials and is excluded from Git via `.gitignore` — **never commit it**
- The server is not exposed to the public internet — Tailscale restricts access to authenticated devices on your tailnet only
- The `/fetch-csv` and `/fetch-private` proxy endpoints use URL whitelists — only IESO report URLs are permitted
- Credentials are loaded server-side at startup and are never sent from or stored in the browser

## Data Sources

| Report | Source |
|---|---|
| DA Hourly Energy LMP | `https://reports-public.ieso.ca/public/DAHourlyEnergyLMP/` |
| Predispatch Hourly Energy LMP | `https://reports-public.ieso.ca/public/PredispHourlyEnergyLMP/` |
| Predispatch Schedules (confidential) | `https://reports.ieso.ca/api/v2.0/files/private/` |

Private report access requires a machine account issued by IESO. See the [IESO Reports Site API Guide](https://www.ieso.ca/) for details.
